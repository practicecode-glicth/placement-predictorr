# placement-predictorr# 🎓 Placement Prediction System

## 🚀 Overview

This project predicts whether a student will get placed based on academic and skill parameters using Machine Learning.

## 🧠 Tech Stack

* Machine Learning: scikit-learn
* Backend: Flask
* Frontend: Streamlit
* CI/CD: GitHub Actions
* Deployment: Render

## ⚙️ Features

* Predict placement chances
* REST API using Flask
* Interactive UI using Streamlit
* Automated CI/CD pipeline

## 🔁 CI/CD Workflow

* Code push triggers GitHub Actions
* Installs dependencies
* Trains model
* Runs tests
* Deploys automatically on Render
